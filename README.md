## Introduccion

Pacman es un juego donde controlas un personaje que se come capsulas.
El personaje principal tiene qie escapar de 4 fantasmas:
----
- Fantasma rojo
- Fantasma Verde
- Fantasma Azul
- Fantas Amarillo

## Controles
- Arriba.
- Abajo.
- Izquierda
- Derecha

## Compilar

Para compilar el proyecto ejecuta el comando:

> **make run**
